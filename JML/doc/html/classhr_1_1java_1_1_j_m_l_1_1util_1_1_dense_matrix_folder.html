Unfolds a vector into matrices by the rules defined in the sizeArray. The sizeArray must have following format: in each row the row and column count must be provided.
Example: sizeArray = {{2,3},{3,2}} will unfold into matrix 0 with 2 rows and 3 columns and matrix 1 with 3 rows and 2 columns.
